# LDS - Backend Contracts

Este proyecto LDS hoy usa `Vite + React` y no define aun backend ni ORM.
Por eso estos archivos se dejan como contrato tecnico inicial para backend, integraciones y futura capa de persistencia.

Contenido:
- `sql/`: migraciones SQL base en estilo PostgreSQL
- `examples/`: respuestas JSON de referencia para frontend e integracion
- `../planning/sprint-05.md`: desglose tecnico del Sprint 5

Objetivo:
- no bloquear frontend por falta de contratos
- alinear negocio, datos y API antes de implementar servidor
- dejar una base versionable dentro del repo
